# ლექცია 1.1 ნპმ ის მომიხილვა
npm შეგვიძლია განვიხილოთ როგორც სასარგებლო მოდულების საცავი ჯავასკრიპტის პროგრამული ენისთვის, მოდულები შექმნილია და მათი

```bash

```
